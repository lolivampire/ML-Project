# 🚀 ML Engineer Journey — 24 Weeks

Dokumentasi perjalanan belajar Machine Learning Engineering dari fundamental
hingga production-grade deployment.

## 🗺️ Roadmap
- **Phase 1** (Week 1–4): ML Fundamentals
- **Phase 2** (Week 5–8): API & Deployment
- **Phase 3** (Week 9–12): Docker & Clean Architecture
- **Phase 4** (Week 13–16): Database & Observability
- **Phase 5** (Week 17–20): System Design & Scale
- **Phase 6** (Week 21–24): Portfolio & Integration

## 📁 Struktur Repository
Setiap folder `week-XX/` berisi:
- `notebooks/` — Eksplorasi dan eksperimen
- `scripts/` — Kode Python yang clean dan reusable
- `data/` — Dataset lokal (tidak di-push)
- `README.md` — Ringkasan topik dan output minggu tersebut

## 🛠️ Tech Stack
Python · Pandas · Numpy · Scikit-learn · FastAPI · Docker · PostgreSQL
